In This repository, you will be able to download the flies to play the Game named : Antivirus.
The goal is simple : lead the red virus out of the board by using up,left,down and right button/key
You can also move other pieces to succeed by pressing the number shown on the pieces : for example you see the number 3 on a piece, you press 3 to move it. Then you finish the goal by pressing the number of the red virus.
There are some white circle that are motionless obstacles. Moreover some circles may have on them 3 red exclamation. Meaning that some pieces cannot move near them.
Finally,i added some music for each level to enhance game experience for users.
I Hope you will enjoy my game.
Good luck !
